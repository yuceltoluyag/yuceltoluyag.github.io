# Security Policy

## Supported Versions

This project is a personal blog site and is continuously kept up-to-date. Security updates and maintenance are performed regularly.

| Component           | Support Status     |
| ------------------- | ------------------ |
| Pelican CMS         | Latest Version     |
| Python Dependencies | Regular Updates    |
| Theme and Plugins   | Regular Updates    |

## Reporting a Vulnerability

If you find a security vulnerability, please contact me using one of the following methods:

*   **Email:** yuceltoluyag@gmail.com
*   **GitHub Issues:** You can report it by opening a new issue in the project's GitHub repository.

Reported vulnerabilities will be investigated as soon as possible, and necessary measures will be taken. You can generally expect a response within 24-48 hours. You will be informed whether the vulnerability is accepted or declined.
